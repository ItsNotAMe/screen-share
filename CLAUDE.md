# Project Memory

This file points contributors and local coding agents at the repo memory files.

Read the shared memory in `agents/` before working:

- `agents/project-memory.md` for current state, preferences, and recent PRs.
- `agents/todo.md` for the ordered roadmap and unfinished work.
- `agents/repo-map.md` for source layout and build shape.
- `agents/ui.md` for Qt desktop UI direction and layout notes.
- `agents/live-pipeline.md`, `agents/av-sync.md`, `agents/adaptation.md`, and `agents/packaging.md` for topic-specific implementation notes.

Keep `agents/` as the source of truth for repo memory. Update those files when useful.
